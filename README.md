# Java Lectures Website

This website provides a comprehensive course of 15 lectures on the fundamentals of Java programming. It is designed for beginners and covers topics from the basics of the language to more advanced concepts like Object-Oriented Programming, Collections, and Design Patterns.

## Design

The website features a modern and immersive design built on the principles of **glasmorphism**. This design aesthetic is characterized by:

*   **Blurred Transparency:** UI elements have a frosted-glass look, creating a sense of depth and hierarchy.
*   **3D Globe Background:** A dynamic, animated 3D globe serves as the background, providing a visually engaging experience without distracting from the content.
*   **Vibrant Colors:** The color palette is clean and modern, with vibrant accent colors that highlight key information and interactive elements.
*   **Responsive Layout:** The design is fully responsive and adapts seamlessly to various screen sizes, ensuring a great user experience on desktops, tablets, and mobile devices.
